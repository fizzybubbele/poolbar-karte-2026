## GitHub Pages aktivieren (einmalig)

1. [Settings → Pages](https://github.com/fizzybubbele/poolbar-karte-2026/settings/pages) öffnen
2. **Build and deployment → Source:** `Deploy from a branch`
3. **Branch:** `main`
4. **Folder:** `/docs` **oder** `/ (root)` — beides ist im Repo vorhanden
5. **Save** — nach 1–2 Minuten sollte die URL erreichbar sein
